About Measure Platform

The Measure platform as bean build in context of Measure ITEA 3 project .The project have for goal to implementing a comprehensive set of tools for automated and continuous measurement,this project have for goal increase the quality and efficiency of software engineering in Europe.

Structured Metrics Metamodel

Measures supported by the platform are defined using the Structured Metrics Metamodel (SMM) provided by the OMG SMM specifies a metamodel for defining, representing and exchanging both measures and measurement information related to any structured information model.
